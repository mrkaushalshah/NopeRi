   - In `background_pipeline`, replace the synchronous loop over companies with a `ThreadPoolExecutor` block:
     ```python
     batch_size = 5 if mode == "production" else 2
     with ThreadPoolExecutor(max_workers=batch_size) as executor:
         futures = [
             executor.submit(process_single_company, company, client, location, radius, db)
             for company in new_companies
         ]
         for future in futures:
             try:
                 result = future.result()
                 if result:
                     print(f"  [Pipeline] Concurrently processed and saved: {result['name']}")
             except Exception as e:
                 print(f"  [Pipeline] Error in concurrent worker: {e}")
     ```
